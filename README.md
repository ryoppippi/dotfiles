# dotfiles

## Requirements

- [rhysd/dotfiles](https://github.com/rhysd/dotfiles)

  `go install github.com/rhysd/dotfiles@latest`

## Usage

``` sh
cd ~
dotfiles clone ryoppippi
```

or

``` sh
curl -L https://raw.githubusercontent.com/ryoppippi/dotfiles/main/setup.sh | sh
```
